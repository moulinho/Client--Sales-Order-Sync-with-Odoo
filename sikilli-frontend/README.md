# sikilli — Next.js Frontend

Frontend for the Sikilli project. See the [root README](../README.md) for full setup instructions, Odoo credentials, and design decisions.

## Quick start (standalone)

```bash
yarn install
yarn dev   # http://localhost:3000
```

Requires the NestJS API running on http://localhost:3001 (calls are proxied via `next.config.ts`).
